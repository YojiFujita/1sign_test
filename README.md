# 1Sign - マッチングアプリ

1人の大家と、住む人のための、マッチングアプリ

## 機能

- **Business 1 Sign**: 建築家やリフォーム業者とのマッチング
- **1Sign Cyber Access Port™**: デジタル鍵管理システム
- **AI × SaaS**: アパート管理機能
- **Interior Database**: 内装データベース

## 技術スタック

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Remix Icon

## 開発環境のセットアップ

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開く

## ビルド

```bash
npm run build
npm start
```

## デプロイ

このプロジェクトはNetlifyにデプロイされます。
